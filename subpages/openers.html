Opener Products

Genie Openers

1028

The 1028 is the base model genie that comes with the option of belt or chain rails. Included are safety eyes, wall button, keypad and remotes. If you are looking for the best cost vs product reliablity this is what you want.

1028B

The 1028B is the same as the base 1028 but includes a battery backup. Very useful if the power goes out and you need the door to still operate for a few days.

2028

This model is one above the 1028. Features are all the same as the 1028 aside from a small power increase with a soft stop and start control feature. Nice to have if you want your door to start and stop it's cycles smoothly.

2028B

Just like the 1028B this is the battery backup version of the 2028 and both units come with the option of belt or chain.

3120

The 3120 is the higher end model we install because of the quality and versatility complimented by the wifi connectivity. Aladdin connect comes standard with no extra accessories which makes this worth the extra cost for a complete opener. Comes in chain or belt drive as well.

3120B

If you would like to add the battery back as well then this is the model for you.

6070 Wall Mount

The 6070 & 6170 are the newest genie openers. This style removes the need for a rail and head opener. They are state of the art and powerful. 6070 comes with all the standard features but doesn't come with the battery backup or aladdin connect.

6170 Wall Mount

In the 6170 the battery backup is included and the aladdin connect is integrated so you don't have to buy extra accessories.

Lift Master

8160W

This opener has basic features but does include built in wifi for myQ control, has a DC motor, works with key by amazon and only comes with chain drive.

8160WB

This is the battery backup version of the base 8160W.

8355W

The 8355W is a belt drive 1/2 HP opener. Its comes with all the same features as the 8160 but comes with more power and the quiet belt drive.

8550WLB

If you want wifi and Battery backup then this is the opener for you. It boasts a handful of features with a friendly price point.

8500 Wall Mount

This is the wall mount unit that elminates the head and rail setup.This model doesn't come with a battery backup but it is optional for purchase. No wifi is included with this unit either. If you are just looking for a basic wall mount setup this is for you.

8500W

This is the upgraded version of the 8500. It comes with wifi and a battery backup. This unit also asthetically looks the best with the gloss finish on the front and comes with the led ceiling light.
